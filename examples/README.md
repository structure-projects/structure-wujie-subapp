# 示例代码目录

这个目录包含了使用 `@structure/wujie-subapp` 的各种示例代码。

## 示例列表

### 1. 基础使用 (`01-basic-usage.ts`)
展示如何最简单地集成和使用这个库：
- 初始化无界子应用
- 向父应用发送数据
- 使用存储管理器
- 清理资源

### 2. 自定义配置 (`02-custom-config.ts`)
展示各种配置选项的使用：
- 自定义存储键名
- 自定义数据同步行为
- 自定义路由配置
- 自定义数据处理钩子
- 完整的自定义配置示例

### 3. 用户 Store (`03-user-store.ts`)
展示如何创建和使用用户 Store：
- 基础用户 Store 创建
- 带自定义处理的用户 Store
- 在组件中使用用户 Store
- 扩展用户 Store

### 4. 权限 Store (`04-permission-store.ts`)
展示如何创建和使用权限 Store：
- 基础权限 Store 创建
- 自定义权限检查逻辑
- 在应用中使用权限 Store
- 与用户 Store 配合使用

### 5. 路由守卫 (`05-router-guard.ts`)
展示如何设置和使用路由守卫：
- 基础路由守卫设置
- 带进度条的路由守卫
- 完全自定义路由守卫
- 渐进式自定义
- 配合面包屑导航

## 如何使用示例

### 1. 直接阅读
每个示例文件都包含详细的注释，您可以直接阅读了解各种用法。

### 2. 在项目中参考
将示例代码复制到您的项目中，根据实际情况进行调整。

### 3. 配合主文档
这些示例应该配合根目录的 `README.md` 一起使用。

## 推荐的学习顺序

1. 先阅读 `01-basic-usage.ts` 了解基础使用
2. 再阅读 `02-custom-config.ts` 了解配置选项
3. 然后根据需要阅读 `03-user-store.ts` 和 `04-permission-store.ts`
4. 最后阅读 `05-router-guard.ts` 了解路由守卫

## 注意事项

- 示例中的模拟 API 和数据仅供演示，实际项目中请替换为真实的 API
- 示例中的 `import` 路径需要根据您的项目结构进行调整
- 某些功能需要配合 Vue 和 Pinia 一起使用
